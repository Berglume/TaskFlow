# TaskFlow

TaskFlow is a small task manager built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as done
- Delete tasks
- Filter by all, active, and done
- Save tasks in localStorage
- Responsive layout

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Run Locally

Open `index.html` in a browser.

## Open in VS Code

1. Open VS Code.
2. Click `File -> Open Folder`.
3. Select the `taskflow` folder.
4. Install the `Live Server` extension if you do not have it.
5. Right-click `index.html`.
6. Click `Open with Live Server`.

Local URL:

```txt
http://127.0.0.1:5502/
```

## Learning Path

1. `index.html` - page structure: form, buttons, list, empty state.
2. `style.css` - layout, colors, spacing, responsive design.
3. `script.js` - variables, events, arrays, rendering, localStorage.

## What This Project Shows

- DOM selection and event handling
- Array methods such as `filter` and `unshift`
- Rendering data into HTML
- Saving and loading data with `localStorage`
- Basic accessibility with labels and semantic HTML
